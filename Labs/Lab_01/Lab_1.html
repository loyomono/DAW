Tecnologias usadas en aplicaciones Web

URL
Uniform Resource Locator, se refiere a las direcciones asignadas a cada recurso que se encuentra por la web, todo esto bajo un mismo esquema.
MIME
Multiporpose Internet Mail Extensions, tambien conocidos como MIME se refiere a todas las especificaciones de dirigidas al intercambio a traves de internet de todo tipo de archivos, de forma transparente para el usuario.
Métodos y códigos de errores HTTP
Existen varias formas por las cuales se obtienen y clasifican los errores de HTTP entre estos se cuentran:
  • Codigos 1XX: Respuestas informativas.
  • Codigos 2XX: Peticiones correctas.
  • Codigos 3XX: Redirecciones.
  • Codigos 4XX: Errores de cliente.
  • Codigos 5XX: Errores de servidor.
HTML5
El HTML5 es la evolucion de las etiquetas HTML, el cual esta pensado para su uso independiente de plugins externos para reproduccion de multimedia, ademas de ser estandarizado y planeado contrario a su antecesor.
JS
JavaScript es un lenguaje de programacion orientado a objetos, se utiliza normalmente del lado del cliente permitiendo mejoras en la interfaz de usuario y paginas dinamicas aunque su uso fuera del lado del cliente es amplio ya que permite el despliegue de aplicaciones web, otras fuera de la web y widgets.
PHP
PHP es un lenguaje de programacion nacido en 1994, con especial adaptabilidad a el desarrollo web y de uso amplio debido a su facil despliegue que permite ser utilizado en la gran mayoria de ssitemas operativos y plataformas.
jsp
Herramienta bastante similar a PHP en cuanto a funciones (como las webs dinamicas y otros) pero con las ventajas de ser una tecnologia basada en Java lo cual permite una gran cantidad de flexibilidad ya que no solo se centra en desarrollo web y puede ser usado en otras areas, llevando a que la portabilidad de las aplicaciones y el tiempo de desarrollo sea menor.
Ajax
Es una tecnica usada en el desarrollo web para la creacion de aplicaciones interactivas, la ventaja es que estas se ejecutan de manera asincrona en el cliente permitiendo cambios y mejorando la interactividad, velocidad y usabilidad de la misma.
